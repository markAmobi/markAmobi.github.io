My Site

June 05, 2015.


On of the early projects given to us by DBC was to create a site where we could upload weekly blog posts among other things. The primary purpose of this was to enable us to practice HTML and CSS which we were just being introduced to at the moment. I thought this was a good challenge and I set out to create something decent. Below, is the first version of what my site looks like:

Yes... I know. It looks wonderful and extremely creative. This was my first attempt and I think it looks good. After looking around and doing a bit of research, I decide to change the color scheme of the site and this is what it became:

While doing some site maintenance after getting a couple of feedbacks, I realized that the navbar and a few other things could be improved, so I made a few changes, and it came out to be this:


Previous Next


After spending so many hours, I thought it looked pretty decent. I was quite satisfied with what I had made because I used just HTML and CSS to achieve all that.
This week, we were asked to access our site to see if we could make it look more professional. Some suggestions were to implement Jekyll, or use Bootstrap to make things look nicer. I had never used any of these so I had to do some research and try to implement something. I decided to use Bootstrap and below is the outcome:


Previous Next


I think it looks alright. Bootstrap is a popular HTML, CSS, and JavaScript framework for making responsive websites and also to make CSS less painful. It made a few things look nicer, and I also decided to change the color once more.
Implementing bootstrap was initially challenging because I had never used it before. I had to do some research to understand the basics. I went over some tutorials from W3schools and I also looked at some samples on the Bootstrap website. After a couple of hours, I was set to apply it to my site. Another challenging part of making changes to my site was going through at least 16 different files and making individual changes to all of them because I was attempting to redesign the whole thing using bootstrap. After hours of testing, and making changes, I was reasonably successful.

The whole process is a good learning experience for me. When I look back at previous versions that I had (thanks to git), I see the progress that I've made through out the different versions. I think it is quite encouraging to know that things are actually progressing and it doesn't matter how small each progression might be.
